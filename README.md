# fashion-mnist-image-classification
A deep learning image classification project using TensorFlow and Keras to classify clothing items from the Fashion MNIST dataset.
#  Fashion MNIST Image Classification using TensorFlow

This project demonstrates a **deep learning image classification model** built using **TensorFlow and Keras**.  
The model is trained on the **Fashion MNIST dataset** to classify images of clothing items into 10 categories.

---

##  Dataset Information

**Fashion MNIST** is a dataset of:
- 70,000 grayscale images
